# Baseline User Flows

## 1. Dashboard
1. Open the dashboard.
2. Review KPI cards for incidents, AI approval rate, search accuracy, and zero-result rate.
3. Open recent knowledge entries from the dashboard list.

## 2. Search
1. Enter a free-text query.
2. Narrow by DBMS, priority, and status.
3. Open a search result and move through previous and next results.

## 3. Knowledge Detail
1. Review symptom, cause, action, runbook, and diagnostic SQL.
2. Submit helpful or not-helpful feedback.
3. Open similar knowledge entries from the sidebar panel.

## 4. Quick Input
1. Select a DBMS and priority.
2. Paste the raw incident content.
3. Generate an AI draft, edit the fields, and save or submit it.

## 5. Reviewer Approval
1. Open the reviewer dashboard.
2. Select a reviewed or needs-review entry.
3. Approve, reject, or bulk-approve entries with version ranges.

## 6. Zero Result Recovery
1. Open the zero-result analysis page.
2. Identify unmatched search queries.
3. Launch Quick Input with the missed query as prefill.
